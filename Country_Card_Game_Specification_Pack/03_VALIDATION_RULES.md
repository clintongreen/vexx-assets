# Validation Rules

- Schema is locked.
- Column order is locked.
- Never invent enum values.
- Every asset URL is populated.
- Missing optional assets use blank.png.
- Descriptions describe the flag only.
- Symbols must visibly appear on the flag.
- Patterns must come from the approved list.
- Category values optimise gameplay.
- Language stores a primary gameplay language.
- Avoid counts where a shared value exists.
- Prefer values useful for quests.
- Every quest value should ideally exist on at least three countries.
